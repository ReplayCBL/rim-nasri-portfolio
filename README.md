**Racing Game Prototype (Unreal Engine 5)**

A 3D racing prototype developed in Unreal Engine 5, focused on building responsive vehicle controls and core gameplay systems.

The project explores how to create a smooth driving experience while implementing essential racing mechanics such as player input, camera behavior, and basic game flow.

---

**Key Features:**

* Vehicle movement system using Unreal Engine tools
* Player input handling and control tuning
* Camera follow system for gameplay clarity
* Basic race flow setup (start / movement / interaction)

---

**Controls**

| Action        | Key |
|--------------|------|
| Accelerate   | W    |
| Brake        | S    |
| Steer        | A / D|
| Handbrake    | Space|
| Camera       | Mouse|

* Movement responsiveness and control feel
